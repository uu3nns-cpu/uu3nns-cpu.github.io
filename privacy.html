개인정보 보호정책

최종 업데이트: 2025년 11월 9일

1. 개요

RE: Report Effortless("서비스")는 사용자의 개인정보를 중요하게 생각합니다. 본 개인정보 보호정책은 서비스가 수집, 사용, 공유하는 정보와 사용자의 권리에 대해 설명합니다.

2. 수집하는 정보

2.1 자동으로 수집되는 정보

서비스는 다음과 같은 정보를 자동으로 수집할 수 있습니다:

  • 사용 데이터: 페이지 조회수, 클릭 수, 사용 시간 등
  • 기기 정보: 브라우저 유형, 운영 체제, IP 주소
  • 쿠키 및 유사 기술: 사용자 경험 개선을 위한 쿠키 사용

2.2 사용자가 제공하는 정보

서비스는 다음과 같은 정보를 저장합니다 (모두 로컬 저장소에만 보관):

  • API 키: Groq 및 GPT API 키 (암호화되어 브라우저에만 저장)
  • 보고서 내용: 작성한 상담 메모 및 생성된 보고서
  • 설정 정보: 사용자가 선택한 보고서 설정 및 프리셋

중요: 위 정보는 사용자의 브라우저 로컬 저장소에만 저장되며, 서버로 전송되거나 외부에 공유되지 않습니다.

3. 정보 사용 방식

수집된 정보는 다음 목적으로만 사용됩니다:

  • 서비스 제공 및 유지
  • 사용자 경험 개선
  • 기술적 문제 해결
  • 사용 통계 분석 (Google Analytics 사용)
  • 광고 표시 (Google AdSense 사용 시)

4. 제3자 서비스

4.1 Google Analytics

서비스는 Google Analytics를 사용하여 사용 통계를 수집합니다. Google Analytics는 쿠키를 사용하여 사용자의 서비스 이용 방식을 분석합니다.

Google의 개인정보 보호정책: https://policies.google.com/privacy

4.2 Google AdSense

서비스는 광고를 표시하기 위해 Google AdSense를 사용합니다.

Google AdSense가 수집하는 정보:

  • 쿠키 및 웹 비콘: 사용자의 관심사 및 웹사이트 방문 기록
  • 기기 정보: IP 주소, 브라우저 유형, 운영 체제
  • 광고 상호작용: 광고 조회수, 클릭수

개인화 광고 및 사용자 선택권:

Google은 사용자의 관심사에 맞는 광고를 표시하기 위해 쿠키를 사용합니다. 사용자는 다음 방법으로 개인화 광고를 제어할 수 있습니다:

  • Google 광고 설정: https://www.google.com/settings/ads
  • 광고 정보 센터: https://www.google.com/ads/preferences/
  • Google의 개인정보 보호정책: https://policies.google.com/privacy
  • Google의 광고 사용 방식: https://policies.google.com/technologies/ads

중요: 광고를 비활성화하더라도 광고는 계속 표시되지만, 개인화되지 않은 광고가 표시됩니다.

4.3 외부 API 서비스

서비스는 AI 보고서 생성을 위해 다음 외부 API를 사용합니다:

  • Groq API: 사용자가 입력한 상담 메모를 처리
  • OpenRouter API (GPT-4o-mini): 사용자가 입력한 상담 메모를 처리

중요: API 키는 사용자가 직접 발급하고 관리하며, 상담 메모는 보고서 생성 목적으로만 API에 전송됩니다. 서비스 제공자는 API 키나 상담 내용을 저장하거나 접근하지 않습니다.

5. 데이터 저장 및 보안

5.1 로컬 저장

모든 사용자 데이터(API 키, 보고서, 설정)는 브라우저의 로컬 저장소(localStorage)에만 저장됩니다.

  • API 키는 AES-256 암호화로 보호됩니다
  • 데이터는 사용자의 기기에서만 접근 가능합니다
  • 서비스 제공자는 사용자 데이터에 접근할 수 없습니다

5.2 데이터 백업

사용자는 서비스 내 "데이터 관리" 페이지에서 자신의 데이터를 백업하고 복원할 수 있습니다. 백업 파일은 사용자의 기기에만 저장됩니다.

6. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 접근권: 브라우저 개발자 도구를 통해 저장된 데이터 확인 가능
  • 수정권: 설정 페이지에서 언제든지 데이터 수정 가능
  • 삭제권: 데이터 관리 페이지에서 언제든지 데이터 삭제 가능
  • 이동권: 백업 기능을 통해 데이터 내보내기 가능

7. 쿠키 정책

서비스는 다음과 같은 쿠키를 사용합니다:

  • 필수 쿠키: 서비스 제공에 필수적인 쿠키
  • 분석 쿠키: Google Analytics를 통한 사용 분석
  • 광고 쿠키: Google AdSense를 통한 맞춤 광고 (해당 시)

대부분의 브라우저는 쿠키를 자동으로 수락하지만, 브라우저 설정에서 쿠키를 거부하거나 삭제할 수 있습니다. 단, 쿠키를 비활성화하면 일부 기능이 제한될 수 있습니다.

8. 아동의 개인정보

서비스는 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다. 만약 아동의 개인정보가 수집된 것을 발견하면 즉시 삭제합니다.

9. 개인정보 보호정책 변경

본 개인정보 보호정책은 필요에 따라 업데이트될 수 있습니다. 변경 사항은 본 페이지에 게시되며, 중요한 변경의 경우 서비스 내 공지사항으로 안내합니다.

10. 문의

개인정보 보호정책에 대한 문의사항이 있으시면 서비스 내 "의견 제출" 기능을 이용해주세요.

11. 동의

서비스를 사용함으로써 사용자는 본 개인정보 보호정책에 동의하는 것으로 간주됩니다.
