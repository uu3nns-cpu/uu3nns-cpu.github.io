API 키 설정

1. Groq API 키 발급

  1. console.groq.com 접속
  2. Google/GitHub 로그인
  3. API Keys → Create API Key
  4. 키 복사 (gsk_로 시작)

2. GPT-4o-mini API 키 발급

  1. openrouter.ai 접속
  2. Google 계정 로그인 (무료 $5)
  3. Keys → Create Key
  4. 키 복사 (sk-or-v1-로 시작)
