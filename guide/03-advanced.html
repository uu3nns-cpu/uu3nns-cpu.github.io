AI 챗봇과 보고서 관리

💬 AI 챗봇

사용법: 오른쪽 하단 🤖 버튼 클릭 (단축키: Ctrl+B)

📝 활용 예시

  • 문체 변경: "문장을 더 공식적으로 바꿔줘"
  • 내용 추가: "2번 문단에 긍정적 변화 추가해줘"
  • 내용 수정: "마지막 문단을 100자로 줄여줘"
  • 설명 요청: "전문 용어를 쉽게 풀어서 설명해줘"

💡 효과적인 사용법

  • 구체적으로 요청 ("100자 추가")
  • 단계적으로 수정
  • 예시 제공
  • 마음에 안 들면 재요청

📚 보고서 관리

저장 방식

  • 자동 저장: 작성 중 임시 저장
  • 작업 저장: 최대 50개 보관 (Ctrl+S)

⚠️ 주의

  • 브라우저 캐시 삭제 시 모두 사라짐
  • 50개 초과 시 오래된 것부터 자동 삭제
  • 중요한 보고서는 TXT 다운로드 필수!

🎯 주요 기능

  • 검색 & 필터: 제목/내용/기간별 검색
  • 상담 일시: 실제 상담 날짜/시간 기록
  • 내보내기: TXT 파일로 다운로드
  • 편집/삭제: 카드에서 직접 실행

🎯 추천 작업 흐름

  1. 메모 작성 (핵심만 간단히)
  2. 보고서 생성 (Ctrl+Enter)
  3. 챗봇 수정 (Ctrl+B)
  4. 즉시 저장 (Ctrl+S)
  5. 정기 백업 (TXT 다운로드)

💡 Pro 팁

  • 비교: 같은 메모로 여러 버전 생성 후 선택
  • 정리: 상담 일시 바로 기록하는 습관
  • 관리: 주기적으로 불필요한 보고서 삭제
