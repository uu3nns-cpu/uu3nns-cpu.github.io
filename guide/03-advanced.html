🎨 고급 기능

AI 챗봇, 히스토리 관리 등 심화 기능을 배웁니다

🎯 목표: 효율성 극대화

💬 AI 챗봇으로 보고서 수정하기

보고서를 생성한 후 "이 부분만 바꿔줘" 같은 요청을 할 수 있는 기능입니다. 직접 수정하는 대신 AI에게 말로 지시할 수 있어요!

🤖
챗봇 사용법
보고서 작성 페이지 오른쪽 하단의 🤖 챗봇 버튼을 클릭하면 챗봇 창이 열립니다. [닫기]는 챗봇 우측 상단

📝 챗봇 활용 예시

💬 사용자 요청
"문장을 좀 더 공식적으로 바꿔줘"
💬 사용자 요청
"2번 문단에 '긍정적 변화'에 대한 내용을 추가해줘"
💬 사용자 요청
"현재 결과는 유지하면서 사례개념화를 추가로 400자 정도로 작성해줘"

💡 챗봇 활용 팁

  • 구체적으로: "좀 더 길게" 보다는 "100자 정도 추가해줘"
  • 단계적으로: 한 번에 여러 요청보다 하나씩
  • 예시 제공: "이런 식으로 바꿔줘" 하고 예시를 보여주기
  • 실험 정신: 마음에 안 들면 다시 요청하기
AI 챗봇 인터페이스

📚 히스토리 관리하기

RE는 최근 작성한 보고서를 최대 50개까지 자동 저장합니다. 언제든 불러와서 수정하거나 복사할 수 있어요.

히스토리 기능

💾
일시 자동 저장
메모 작성 중, 다른 페이지로 넘어가거나 하더라도 작성 중 메모를 불러올 수 있습니다.

작업 저장을 버튼을 눌러 메모와 함께 결과를 저장할 수 있으며 작업보기를 통해 언제든지 불러올 수 있습니다.

단, 브라우저 데이터를 삭제하면 히스토리도 사라집니다
🔍
검색 및 필터
날짜별, 키워드별로 이전 보고서를 빠르게 찾을 수 있습니다.
📤
내보내기 / 불러오기 (미구현 기능)
내보내기: 히스토리 전체를 JSON 파일로 백업
불러오기: 다른 컴퓨터나 브라우저로 히스토리 이동

히스토리 활용법

  1. 대시보드에서 "📋 최근 작성 히스토리" 섹션 확인
  2. 원하는 보고서 클릭
  3. 내용 확인 후:
    • 수정: 보고서 작성 페이지에서 다시 열기
    • 복사: 클립보드로 복사
    • 삭제: 불필요한 항목 제거

⚠️ 백업 권장

중요한 보고서는 정기적으로 "내보내기" 기능으로 백업하세요. 브라우저 캐시를 지우거나 다른 컴퓨터로 이동할 때 유용합니다.

⌨️ 단축키로 빠르게 작업하기

자주 사용하는 기능을 키보드로 빠르게 실행할 수 있습니다.

기능 단축키
보고서 생성 Ctrl + Enter
보고서 복사 Ctrl + Shift + C
저장 Ctrl + S
챗봇 열기/닫기 Ctrl + B

💡 단축키 팁 (미구현 기능)

설정 페이지에서 단축키를 커스터마이징할 수 있습니다!

📋 템플릿 활용하기

자주 사용하는 보고서 형식을 템플릿으로 저장해두면 매번 처음부터 작성할 필요가 없습니다.

템플릿 만들기

  1. 마음에 드는 보고서 작성
  2. "템플릿으로 저장" 버튼 클릭
  3. 템플릿 이름 입력 (예: "개인상담 보고서")
  4. 저장 완료!

템플릿 사용하기

  1. 보고서 작성 페이지에서
  2. "템플릿 불러오기" 버튼 클릭
  3. 원하는 템플릿 선택
  4. 필요한 부분만 수정
📋 템플릿 예시
개인상담 보고서 템플릿:
- 회기 정보
- 주요 호소 문제
- 상담 내용
- 내담자 반응
- 다음 계획
