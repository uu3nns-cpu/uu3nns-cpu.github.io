✍️ 기본 사용법

API 키 발급부터 첫 보고서 작성까지 알아보기

🎯 목표: 첫 보고서 완성

🔑 1단계: AI 열쇠(API 키) 받기

RE: 가 AI를 사용하려면 "열쇠"가 필요합니다. 이 열쇠를 API 키라고 부릅니다. RE: 는 두 가지 AI 모델 비교 서비스를 지원합니다.

💡 API 키란?

AI에게 "이 사람은 허가받은 사용자입니다"라고 증명하는 비밀번호 같은 것입니다. 신용카드처럼 본인만 알고 있어야 하며, 절대 남에게 알려주면 안 됩니다!

✅ 결제수단 미등록시 무료 사용 가능
✅ 보고서 품질이 더 우수함

단점:
⚠️ 결제수단 등록 시 사용량에 따라 과금 가능
⚠️ 크레딧 소진 시 새 계정 등록 필요

🚀 Groq

장점:
✅ 완전 무료로 사용 가능
✅ 매우 빠른 응답 속도
✅ 간단한 가입 절차

단점:
⚠️ 하루 사용량 제한 있음
⚠️ 보고서 품질이 GPT-4o-mini보다 낮음

🎯 초보자 추천

처음 사용하신다면 GPT-4o-mini (OpenRouter)를 추천합니다! Google 계정으로 쉽게 시작할 수 있고, 첫 이용시 $5 크레딧으로 수천 장의 보고서를 작성할 수 있습니다.

🚀 Groq 및 GPT-4o-mini API 키 발급하기

1 웹사이트 접속

웹브라우저에서 아래 주소로 이동합니다:

🚀 Groq

https://console.groq.com

또는 구글에서 "groq console" 검색

🚀 OpenRouter

https://openrouter.ai/

또는 구글에서 "openrouter" 검색

2 회원가입 / 로그인

오른쪽 위의 "Sign In" 또는 "Sign Up" 버튼 클릭

  • Google 계정으로 간편 가입 가능
  • 이메일로도 가입 가능
OpenRouter(추천) 및 Groq 로그인 화면
3 API 키 생성

🚀 Groq

로그인 후:

  1. "API Keys" 메뉴 클릭
  2. "Create API Key" 버튼 클릭
  3. 키 이름 입력 (예시: "RE-Report")
  4. "Submit" 클릭

🚀 OpenRouter(GPT-4o-mini)

로그인 후:

  1. "프로필 사진" 또는 이름 클릭
  2. 펼침 메뉴에서 "Keys" 클릭
  3. "Create API Key" 버튼 클릭
  4. 키 이름 입력 (예시: "RE-Report")
  5. "Create" 클릭
4 API 키 복사

⚠️ 중요!

API 키는 한 번만 보여집니다! 복사에 실패해도 다시 발급 받을 수 있습니다.

생성된 API 키가 화면에 나타납니다:

🚀 Groq

gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxx...

"Copy" 버튼을 눌러 클립보드에 복사합니다.

🚀 OpenRouter

sk-or-v1-xxxxxxxxxxxxxxxxxxxxxxxxxxx...

"Copy" 아이콘을 눌러 클립보드에 복사합니다.

⚙️ RE: 에 API 키 등록하기

1 설정 페이지 열기

RE 대시보드에서 오른쪽 위의 ⚙️ 설정 버튼 클릭

RE 설정 페이지
2 API 제공자 선택

"API 키 설정" 섹션에서 사용하려는 서비스를 선택합니다:

  • Groq를 사용하는 경우: "Groq API 키" 입력란 클릭
  • OpenRouter(GPT-4o-mini)를 사용하는 경우: "OpenRouter API 키" 입력란 클릭

💡 두 서비스를 모두 등록해두면 언제든지 전환하여 사용할 수 있습니다.

3 API 키 입력

앞서 복사한 API 키를 해당 서비스의 "API 키" 입력란에 붙여넣기 (Ctrl+V)

💡 API 키 형식 예시

  • Groq: gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxx...
  • OpenRouter: sk-or-v1-xxxxxxxxxxxxxxxxxxxxxxxxxxx...
4 저장하기

페이지 아래의 "💾 설정 저장" 버튼 클릭

✅ "설정이 저장되었습니다" 메시지가 나타나면 성공!

📝 2단계: 첫 보고서 작성하기

이제 준비가 끝났습니다! 실제로 보고서를 작성해봅시다.

1 보고서 작성 페이지 열기

대시보드에서 ✍️ 보고서 작성 버튼 클릭

보고서 작성 페이지
2 상담 메모 작성

"상담 메모" 텍스트 영역에 상담 내용을 자유롭게 작성합니다.

💡 작성 팁

  • 평소 메모하듯 편하게 작성하세요
  • 문법이나 맞춤법은 신경 쓰지 않아도 됩니다
  • 대화 형식으로 작성해도 좋습니다
  • 중요한 내용만 간략하게 적어도 충분합니다
✏️ 작성 예시
오늘 내담자와 5회기 상담 진행. 요즘 학교생활이 힘들다고 함.
친구들과의 관계에서 오해가 생겨 스트레스를 많이 받는 것 같음.
자신의 감정을 표현하는 것을 어려워하는 모습.
다음 시간에는 감정 표현 연습을 해보기로 함.
3 보고서 생성

메모 작성을 마쳤으면 "보고서 생성" 버튼 클릭

잠시 기다리면 (약 5~10초) 아래쪽에 보고서가 자동으로 생성됩니다!

생성된 보고서 예시
4 보고서 확인 및 수정

생성된 보고서를 확인합니다:

  • 전문적인 용어로 변환되었나요?
  • 문장이 매끄럽게 연결되었나요?
  • 내용이 정확한가요?

만약 수정이 필요하다면:

  • 오른쪽 하단 AI 챗봇 🤖버튼을 직접 클릭해서 수정 요청
  • 대화하듯 요청사항을 입력하면 부분/전체 수정이 가능합니다.
5 보고서 저장 및 내보내기

마음에 드는 보고서가 완성되면:

  • 📋 복사: 보고서를 클립보드에 복사 (Ctrl+C)
  • 💾 저장: RE:(사용자 컴퓨터)에 저장 (나중에 다시 불러올 수 있음)

🎯 실전 사용 팁

✅ 이렇게 사용하세요

  • 간단 명료하게: "오늘 상담에서 이런 이야기를 나눴어요"
  • 핵심만: 완성도는 AI가 도와줄테니 의식의 흐름대로 적어봐요
  • 재시도, 수정요청: 작성하기를 다시 누르면 조금 다르게 작성돼요

❌ 이런 것은 피하세요

  • 개인식별정보: 실명, 주민번호, 연락처는 최소화
  • 너무 길게: 7천자 제한을 두었어요. 한번에 너무 길지 않게
  • 상충된 요청: 사전설정(간략), 나만의 작성규칙에서 구체적으로

💡 꿀팁

첫 번째 생성 결과가 마음에 안 들면 "보고서 생성" 버튼을 다시 눌러보세요! AI가 매번 조금씩 다르게 생성합니다.
