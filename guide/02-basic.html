📝 단계별 가이드

1 웹사이트 접속

OpenRouter: https://openrouter.ai

Groq: https://console.groq.com

2 Google 계정으로 로그인

"Sign in with Google" 클릭

3 API 키 생성

OpenRouter: 프로필 → Keys → Create API Key

Groq: API Keys → Create API Key

4 API 키 복사

"Copy" 버튼 클릭 (한 번만 표시됨)

⚙️ 3단계: RE: 에 등록

  1. RE 대시보드에서 ⚙️ 설정 클릭
  2. OpenRouter 또는 Groq API 키 입력란에 붙여넣기
  3. 💾 설정 저장 클릭
  4. "설정이 저장되었습니다" 메시지 확인

📝 4단계: 첫 보고서 작성

🎬 보고서 작성 예제 영상

📝 단계별 가이드

  1. ✍️ 보고서 작성 버튼 클릭
  2. 상담 메모 입력 (자유롭게 작성)
  3. 보고서 생성 버튼 클릭 (또는 Ctrl+Enter)
  4. 5~10초 후 보고서 완성!
  5. 필요시 AI 챗봇(🤖)으로 수정
  6. 💾 저장 또는 📋 복사
