🔧 문제 해결

자주 발생하는 문제와 해결 방법을 알아봅니다

🎯 빠른 해결

🔑 API 키 관련 문제

❌ "API 키가 유효하지 않습니다"
▼
원인: API 키를 잘못 입력했거나 만료되었습니다.
해결 방법:
  1. API 키를 다시 복사해서 붙여넣기
  2. 앞뒤 공백이 없는지 확인
  3. Groq / OpenRouter 사이트에서 키가 활성화되어 있는지 확인
  4. 필요시 새 키를 발급받아 다시 등록
  5. Browser 호환성 문제일 수 있어요. Chrome 사용을 권장
❌ "일일 사용량을 초과했습니다"
▼
원인: Groq의 무료 일일 한도를 초과했습니다.
해결 방법:
  • 방법 1: 사용량이 리셋 될때까지 기다리기
  • 방법 2: OpenRouter로 전환 (유료이지만 제한 없음)
  • 방법 3: 다른 Groq 및 OpenRouter 계정 만들기
❓ API 키를 잃어버렸어요
▼
API 키는 한 번만 표시됩니다. 잃어버렸다면:
  1. Groq/OpenRouter 콘솔 접속
  2. 기존 키 삭제
  3. 새 키 발급
  4. RE: 설정에 새 키 등록

📝 보고서 생성 문제

❌ 보고서가 생성되지 않아요
▼
가능한 원인:
  1. 인터넷 연결 확인: Wi-Fi나 네트워크가 연결되어 있나요?
  2. API 키 확인: 설정에서 API 키가 올바른지 확인
  3. 메모 길이: 너무 길거나 짧지 않은지 확인 (50자 이상 권장)
  4. 브라우저 새로고침: F5 키를 눌러 페이지 새로고침
❌ 생성된 보고서가 이상해요
▼
원인: AI가 메모를 잘못 이해했을 수 있습니다.
해결 방법:
  • 메모를 더 구체적으로 작성
  • 핵심 내용을 먼저 작성
  • "보고서 생성" 버튼을 다시 클릭 (매번 다르게 생성됨)
  • AI 챗봇으로 수정 요청
❓ 생성 속도가 너무 느려요
▼
Groq 사용 시: 일반적으로 5~10초 내에 생성됩니다. 더 오래 걸린다면 인터넷 속도를 확인하세요.

OpenRouter 사용 시: 모델에 따라 10~30초 걸릴 수 있습니다.

💾 데이터 관련 문제

❌ 히스토리가 사라졌어요
▼
원인: 브라우저 캐시가 삭제되었거나 다른 브라우저를 사용 중입니다.
해결 방법:
  • 백업 파일이 있다면 "불러오기" 기능 사용
  • 앞으로는 정기적으로 "내보내기"로 백업
  • 중요한 보고서는 별도 파일로 저장
❓ 히스토리를 다른 컴퓨터로 옮기고 싶어요
▼
  1. 기존 컴퓨터에서: 설정 → "히스토리 내보내기" → JSON 파일 저장
  2. 파일을 USB나 클라우드로 새 컴퓨터에 복사
  3. 새 컴퓨터에서: 설정 → "히스토리 불러오기" → 파일 선택
❓ 저장 용량이 부족하다고 나와요
▼
RE: 는 브라우저 저장공간을 사용합니다. 용량이 부족하다면:
  • 오래된 히스토리 삭제
  • 브라우저 캐시 정리
  • 다른 웹사이트의 데이터 삭제

🌐 브라우저 관련 문제

✅ 권장 브라우저

RE는 다음 브라우저에서 가장 잘 작동합니다:

  • Chrome (버전 90 이상) - 가장 권장
  • Edge (버전 90 이상)
  • Firefox (버전 88 이상)
  • Safari (버전 14 이상)
❓ 화면이 깨져 보여요
▼
  1. 브라우저를 최신 버전으로 업데이트
  2. Ctrl + F5로 강력 새로고침
  3. 브라우저 확대/축소 비율을 100%로 설정 (Ctrl + 0)
  4. 다른 브라우저로 시도
❓ 모바일에서도 사용할 수 있나요?
▼
네! RE: 는 모바일에서도 작동합니다. 하지만:
  • 화면이 작아 PC보다 불편할 수 있음
  • 키보드 입력이 많으므로 태블릿 권장
  • 인터넷 데이터 사용량 확인 (Wi-Fi 사용 권장)

❓ 자주 묻는 질문 (FAQ)

💰 비용이 얼마나 드나요?
▼
Groq 사용 시: 완전 무료입니다! 단, 하루 사용량 제한이 있습니다.

OpenRouter 사용 시: 결제 수단을 등록하지 않으면, 제한량까지 무료 사용이 가능합니다.
🔒 제 데이터가 안전한가요?
▼
대부분의 데이터는 사용자 컴퓨터에만 저장됩니다. 보고서 생성 시에만 AI 서버로 임시 전송되며, 처리 후 즉시 삭제됩니다. 자세한 내용은 4. 보안 & 개인정보 페이지를 참고하세요.
👥 여러 명이 함께 사용할 수 있나요?
▼
현재 RE: 는 개인 사용을 위해 설계되었습니다. 여러 명이 사용하려면:
  • 각자 다른 API 키 사용 (API키를 공유하는 경우 사용량을 금방 소진할 수 있음. 주의!)
  • 각자 다른 브라우저 프로필 사용
  • 또는 각자 RE: 폴더를 별도로 복사
📱 오프라인에서도 사용할 수 있나요?
▼
아니요, RE: 는 AI 서버와 통신이 필요하므로 인터넷 연결이 필수입니다. 메모 작성은 오프라인에서도 가능하지만, 보고서 생성은 온라인이어야 합니다.
🔄 업데이트는 어떻게 하나요?
▼
새 버전이 나오면:
  1. 기존 히스토리와 API키를 "내보내기"로 백업
  2. 새 버전 압축 파일 다운로드
  3. 기존 폴더 삭제 후 새 파일 압축 해제
  4. 히스토리 "불러오기"로 복구
💬 더 궁금한 점이 있으신가요?
위 내용으로 해결되지 않았다면 개발자에게 문의하세요.
