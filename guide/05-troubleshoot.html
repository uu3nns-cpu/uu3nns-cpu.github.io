문제 해결

자주 발생하는 문제와 해결 방법

🔑 API 키 문제

❌ "API 키가 유효하지 않습니다"
해결:
  1. API 키 다시 복사 후 붙여넣기
  2. 앞뒤 공백 확인
  3. Groq/OpenRouter에서 키 활성화 확인
  4. 필요시 새 키 발급
  5. Chrome 브라우저 사용 권장
❌ "일일 사용량 초과"
Groq 무료 한도 초과 시:
  • 사용량 리셋 대기
  • OpenRouter로 전환
  • 새 계정 생성
❓ API 키를 잃어버렸어요
해결: 기존 키 삭제 → 새 키 발급 → RE 설정에 재등록

📝 보고서 생성 문제

❌ 보고서가 생성되지 않아요
확인 사항:
  1. 인터넷 연결 확인
  2. API 키 올바른지 확인
  3. 메모 길이 (50자 이상 권장)
  4. F5로 페이지 새로고침
❌ 생성된 보고서가 이상해요
해결:
  • 메모를 더 구체적으로 작성
  • "보고서 생성" 다시 클릭
  • AI 챗봇으로 수정 요청

💾 데이터 문제

❌ 히스토리가 사라졌어요
원인: 브라우저 캐시 삭제 또는 다른 브라우저 사용
해결: 백업 파일 "불러오기" 또는 정기적으로 TXT 다운로드
❓ 다른 컴퓨터로 옮기고 싶어요
방법:
  1. 설정 → "히스토리 내보내기" → JSON 저장
  2. 새 컴퓨터로 파일 복사
  3. 설정 → "히스토리 불러오기"

🎬 동영상이 재생되지 않을 때

❌ 유튜브 동영상이 보이지 않아요
확인 사항:
  1. 인터넷 연결 확인 - 동영상은 온라인에서만 재생됩니다
  2. 브라우저 업데이트 - Chrome, Edge 최신 버전 권장
  3. 광고 차단기 - 임시로 비활성화 후 시도
  4. F5 새로고침 - 페이지를 다시 로드해보세요
  5. 다른 브라우저 - Chrome, Edge, Firefox 중 다른 것 시도
그래도 안 되면:
  • 유튜브에서 직접 영상 URL 접속해보기
  • 영상이 업로더에 의해 비공개/삭제되었을 수 있음
❌ "동영상 소유자가 외부 재생을 차단했습니다" 메시지
원인: 해당 영상의 업로더가 외부 사이트 재생을 차단한 경우
해결: 유튜브에서 직접 시청하거나, 관련 대체 영상 찾기
❓ API 발급 영상이 안 보여요 (설정 페이지)
해결:
  1. 설정 페이지 새로고침 (F5)
  2. "발급방법" 버튼 → 문서로 확인
  3. 직접 유튜브 링크로 접속:
    • Groq API 발급 영상
    • GPT API 발급 영상

🌐 브라우저 문제

✅ 권장 브라우저

  • Chrome (버전 90+) - 가장 권장
  • Edge (버전 90+)
  • Firefox (버전 88+)
  • Safari (버전 14+)
❓ 화면이 깨져요
해결:
  1. 브라우저 최신 버전으로 업데이트
  2. Ctrl+F5로 강력 새로고침
  3. 확대/축소 100% (Ctrl+0)
  4. 다른 브라우저 시도

❓ 자주 묻는 질문

💰 비용이 얼마인가요?
Groq: 완전 무료 (하루 사용량 제한)
OpenRouter: 결제수단 미등록 시 제한량까지 무료
🔒 데이터가 안전한가요?
대부분 데이터는 사용자 컴퓨터에만 저장. AI 전송 데이터는 즉시 삭제. 보안 가이드 참고
👥 여러 명이 사용 가능한가요?
각자 다른 API 키 또는 다른 브라우저 프로필 사용 권장
📱 오프라인 사용 가능한가요?
불가능. AI 서버 통신을 위해 인터넷 연결 필수
