💡 핵심

대부분의 데이터는 여러분의 컴퓨터에만 저장됩니다. AI 서버로 전송되는 데이터는 처리 후 즉시 삭제됩니다.

🛡️ 왜 안전한가?

핵심 원리: 중앙 서버 없음

RE는 네이버, 카카오처럼 중앙 서버에 데이터를 모으지 않습니다. 여러분의 컴퓨터에서 직접 AI 서비스를 호출합니다.

3가지 안전 장치

  1. RE 서버는 데이터를 보지도, 저장하지도 못함
  2. AI 서버도 처리 후 즉시 삭제 (GDPR 준수)
  3. 여러분만 데이터를 소유하고 통제

✅ 결론

RE는 데이터를 저장할 서버 자체가 없습니다. 해킹이나 유출이 구조적으로 불가능합니다.

✅ 보안 체크리스트

  1. API 키 보호: 절대 공유 금지, 누출 시 즉시 재발급
  2. 공용 PC 주의: 사용 후 브라우저 캐시 삭제 또는 시크릿 모드 사용
  3. 개인정보 최소화: 실명, 주민번호, 연락처 최소화
  4. 정기 백업: 중요 보고서는 TXT 다운로드
  5. 안전한 네트워크: 공공 Wi-Fi 대신 안전한 네트워크 사용

❓ 자주 묻는 질문

Q1. AI 서버에서 정말 즉시 삭제되나요?

네. Groq와 OpenRouter는 GDPR, HIPAA 준수 기업으로 API 호출 완료 후 데이터를 즉시 삭제합니다.

Q2. API 키가 유출되면?

  • Groq: 무료 한도 소진 (금전적 손실 없음)
  • OpenRouter: 사용료 청구 (금전적 손실 발생)

해결: 즉시 해당 서비스에서 키 삭제 후 재발급

Q3. 기관 보안 정책 위배 여부는?

각 기관 정책에 따라 다릅니다. 사용 전 기관 담당자에게 문의하세요.

Q4. 브라우저 데이터 삭제 시?

히스토리와 설정이 모두 사라집니다. 중요 데이터는 반드시 백업하세요.

⚖️ 권장 vs 금지

✅ 권장

  • 익명 표현 ("내담자 A")
  • 개인 컴퓨터 사용
  • 정기적 백업
  • 안전한 네트워크

❌ 금지

  • 실명, 주민번호 기재
  • 공용 PC 무분별 사용
  • API 키 공유
  • 공공 Wi-Fi 사용
